# Tip Calculator

A basic tip calculator to compute tip and total amount based on the bill.

## Features

- Input bill amount
- Choose predefined tip percentages (5%, 10%, 15%, 25%, 50%)
- View calculated tip and total amount
- Reset functionality

## Tech Stack

- HTML
- CSS
- JavaScript
